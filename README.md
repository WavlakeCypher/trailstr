# TrailStr\n\nA trail & activity social app built on Nostr. Strava + AllTrails, decentralized.\n\n## Tech Stack\n- React 18 + TypeScript + Vite\n- Tailwind CSS\n- MapLibre GL JS\n- Nostr protocol (nostr-tools)\n- Recharts\n- Zustand\n\n## Development\n\n```bash\nnpm install\nnpm run dev\n```\n\n## Architecture\n\nZero backend. All data lives on Nostr relays. All computation is client-side.\n
